## Installation

RAVAnimator is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'RAVAnimator', :git=>'https://github.com/K-Be/RAVAnimator', :tag=>'0.1'
```
